You will get token smart contract, presale token smart contract. So, you can create token contract and presale contract for your own custom token.

You can watch video tutorial for following steps.

Create contract files using Remix IDE.

Firstly, you have to go using web browser to https://remix.ethereum.org/ . Create new files. Copy token contract and presale contract from provided folder and paste it on new created files in remix IDE.

Compile contract

You have to compile contract before deploying it. Go to compiler tab section. Select the latest compiler version.

Click over Compile button. Then, Contract will be compiled.

You have to keep abi json for interacting with smart contract using react app.

Deploy Contract

Go to Deploy tab section. Select injected provider metamask for connecting metamask wallet into remix IDE.

You have to select desired network for deploying contract. You can select any network from avalable list.

Deploy token smart contract.

Click over Deploy button. Then, you have to click over Confirm button in metamask popup.

Deploy presale smart contract

You have to set token address, seller address before deploying presale smart contract. In _token field, paste token contract address. In _seller field, paste token contract owner address(your address because you have already deployed token contract using above steps).

Approve token for presale contract

You have to do this step in token contract. Click approve function. In spender field, paste presale token contract address. In amount field, paste token amount(wei format) to approve. Then have to click over transact button. Then have to set approve token amount and click over "Next" button in metamask popup. Finaly, click over Confirm button in metamask popup. Then token amount will be aproved for presale contract.

Update presale token amount in presale contract

You have to do this step in presale token contract. You have to click over "setPresaleTokenAmount" button. Then, click over "Confirm" button in metamask popup. Then token amount will be ready for presale contract.

Add stage information in presale contract

In _bonus field, paste bonus amount between 0 to 100. In _price field, paste price amount(wei format). In _start field, paste start time(unix timestamp). In _end field, paste end time(unix timestamp). Then have to click over transact button. Finaly, click over Confirm button in metamask popup. Repeat this step to add more stage in presale contract.

Contract Configuration

You can watch video instruction for interacting with smart contract using this template.

Add contract in react app

Go to "config.js" file. Paste token contract address for "tokenContractAddress" variable. Paste presale token contract address for "presaleContractAddress" variable.

Add payment method

Go to "config.js" file. Paste payment method. eg. ETH, BNB, MATIC etc.

Add ABI json of contracts

You have already deployed token contract and presale token contract. So, you have kept abi json of those contracts. Now you have to add abi json into react app. Go to "TokenContractAbi.json" and paste token contract abi json. Go to "PresaleContractAbi.json" and paste token contract abi json.

Import ABI json file

Go to "config.js" file. Import "TokenContractAbi.json" and "PresaleContractAbi.json" file.

Interacting with smart contract

Go to "config.js" file.

Interacting with write function of smart contract

Go to "config.js" file. Call public function(buyToken) from presale smart contract.

You can call "buyToken" function from any component file.

You can set onclick event for "buyToken" function to write function on presale contract.

Deploy

Gittu react version Deploy

Gittu React js static build

  • Navigate to gittu_react_main
  • run command npm run build
  • after the export is done you will get a folder called dist. Host that folder anywhere you want
  • to see the export build, run command: npm run dev
    note: images and route will not work in local server

netlify

  • Navigate to gittu_react_main
  • run command:
  • npm run build

    after the build process is done. you will find a new folder called /dist

  • npm install netlify-cli --save-dev
  • netlify login

    after the login process is done

  • run command: netlify deploy
  • choose + Create & configure a new site
  • give you site name
  • path to deploy: ./dist

    after the build process once done, run command:

  • netlify deploy --prod
  • path to deploy: ./dist
  • for more information: How to deploy React Apps in netlify

You will find react.js app deployment complete guide here: React.js app Deployment

vercel

  • Navigate to gittu_react_main

    for deploying to vercel using terminal you need to install vercel-cli on your machine Read more...

  • npm install vercel --save-dev
  • run command:
  • vercel login

    • After the login process is done. run command:

  • vercel

    • after the deploy process is done. then deploy the production version

  • vercel --prod
